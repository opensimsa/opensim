This is the stable version 0.16 with Eskom RT&D themo-mechanical changes added for windows.

These additions are now all part of version 0.17 so Linux user can use the the latest development version 

The x64 version for Win requires Visual C++ Redistributable for VS2013 and supports 64-bit Windows Vista/7/8/10

To use, unpack the .7z archive with 7-zip to an empty folder on your system (avoid using a directory with special characters in the path) then start by double clicking FreeCAD.exe in the bin folder of the extracted folder.

Windows users can now grab latest development release which has far more capabilities

Additional workbenches are available from an add on macro which is part of 0.17 Tools ->  Addon manager
